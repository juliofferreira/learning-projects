# Flexbox course

Alura Flexbox course, discussing the usage of the "display: flex" property on CSS, and its importance for responsive layouts.

All the main flexbox properties and its values were discussed, such as:

- justify-content
- align-items
- align-self
- flex-direction
- flex-wrap
- flex-flow
- order
- flex-grow
- flex-shrink
- flex-basis
- flex
